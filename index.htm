UrlMasher

Creates urls for quickly testing multiple domains built on a common code base.

About

Usage:

  1. Enter your url domain, subdomain and/or path.
  2. Mash them up!
  3. Control-click the links.
  4. Enjoy your new tabs!

Under the hood:

  • LABjs script loader.
  • jQuery.
  • Modernizr/Yepnope.
  • Google Webfonts.
  • HTML5
  • CSS3
  • LocalStorage
  • CSS managed by Compass
